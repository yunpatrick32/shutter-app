# Prompt to Update Cowork from Claude.ai Chat
*Copy and paste this into Cowork whenever you want to sync updates from Claude.ai*

---

## How to Use
1. Have a conversation in Claude.ai about Shutter
2. When you're done, copy the update block below, fill it in, and paste it into Cowork

---

## Update Block (fill this in and paste into Cowork)

```
Hey, I have updates from my Claude.ai chat to sync into the Shutter project. Please update the project memory and SHUTTER_STATUS.md with the following:

**Checklist updates:**
- Completed: [list anything finished]
- Added: [list any new tasks]
- Removed: [list anything no longer relevant]

**Dev updates:**
[paste any new decisions, bug fixes, features discussed]

**Legal updates:**
[paste any brand name decisions, trademark progress, etc.]

**Finance updates:**
[paste any banking, LLC, Stripe, pricing decisions]

**Marketing updates:**
[paste any new contacts, content ideas, outreach results]

**New notes or learnings:**
[anything else worth remembering]

Please update the memory file and refresh SHUTTER_STATUS.md with today's date.
```

---

## Tips
- You don't need to fill in every section — just the ones that changed
- Be as brief or detailed as you want — Cowork will sort it out
- After pasting, Cowork will update both the memory and the status doc automatically
- The status doc can then be re-uploaded to Claude.ai project knowledge to keep it fully in sync
